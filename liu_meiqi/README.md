# 简介
该智能体是基于英语阅读智能体延伸出的针对德语小语种的AI阅读助手。在对话框中，用户在输入一篇专八级别的德语文章后，能够得到以下不同方面的反馈：
1. 中文标题：AI智能体翻译输入的文本的德语标题成中文
2. 中文摘要：AI智能体自行阅读整篇德语文章，形成具体的摘要内容，帮助读者能够快速获得文章内容，作者核心思想等信息。（语言为中文）
3. 德语摘要：AI智能体自行阅读整篇德语文章，形成具体的摘要内容，帮助读者能够快速获得文章内容，作者核心思想等信息。（语言为德语）
4. 3道德语阅读题目：AI智能体在阅读完毕整篇德语文章后，会给用户出3道单项选择题目，旨在检验用户是否已经理解全文以及细节。
5. 思维导图：AI智能体在阅读完毕整篇德语文章后，会以图示的方式给出文章的篇章结构，每一段之间的关联，帮助用户梳理文章脉络，从全局掌握文章内容。用户可以点击该处的链接，获得思维导图。
6. 全文精读：该部分为AI智能体的重要功能展现。通过分段功能，AI智能体能够以段为单位实现精读。精读包含每一段落的翻译，以及其中需要掌握的专八词汇。本智能体新添加的功能：根据德语语言本身的特点给用户补充词汇的词性和复数变化。AI智能体在识别专八考试的词汇过程中，也识别该词汇的类型。如若是名词，则会给出复数和词性的信息。此外，对于词汇学习，除了以上，还包含中文释义，德语例句以及例句翻译。
7. 待学词汇总结：该部分是将所有需要记忆的词汇进行总结，形成个人的单词表。
8. 参考答案：AI智能体针对第四部分的题目给出答案。

注意：如果在看完AI智能体的精读后，若仍对文章内容或是阅读题目的答案有疑问，可以继续追问。在对话框内输入用户的问题，得到智能体的回答。
